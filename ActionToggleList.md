# RotationSolverReborn Action Toggle List

Use `/rotation ToggleActions <Action Name>` to toggle an action that exists in the current rotation.

- Example: `/rotation ToggleActions ファストブレード`
- Explicit enable: `/rotation ToggleActions ノー・マーシー true`
- Explicit disable: `/rotation ToggleActions ノー・マーシー false`
- Matching is case-insensitive, but spaces and punctuation should follow the displayed action name.
- `[PvE]` and `[PvP]` only indicate where that name exists. If the name is the same, the command text is also the same.
- Action names are shown as `Japanese (English)`.
